High Tech, Low Effort


augmented-ui is just CSS.

  • Futuristic, cyberpunk-inspired UI shaping for any element
  • Only requires an attribute (or classes) to equip the augs, and a few settings for each one to make it feel just right

Namespaced to avoid crossing wires.

  • All custom properties begin with "--aug-"
  • Selectors all check for the "augmented-ui" attribute. ("data-" prefix supported too)
  • Prefer class names to the attirbute? Fully supported, all class names begin with "aug-"

Use augmented-ui in any project, for free.
